###  树 Tree 基本信息及实现

![TransitionXmind](https://raw.githubusercontent.com/pro648/tips/master/sources/images/TreePreview.png)

树（Tree）是一种很重要的数据结构，在软件开发的多方面都有使用：

- 表示层级结构。
  - 计算机语言的抽象语法树。
  - 解析人类语言的树。
  - XML 和 HTML 文档对象模型。
  - 处理 JSON 和YAML 文档。
- 快速查找数据。
- 管理有序数据。

树的类型有很多，其有不同的形状和大小。这一篇文章将介绍树的基本信息，以及如何实现树。
