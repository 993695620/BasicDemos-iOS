###  二叉树 Binary Tree

![BinaryTree](https://raw.githubusercontent.com/pro648/tips/master/sources/images/BinaryTreePreview.png)

二叉树（Binary Tree）每个节点至多有两个子节点，被称为左子树（left）、右子树（right）。这个demo涉及以下内容：

- 实现一个二叉树
- 遍历算法
  - 中序遍历
  - 前序遍历
  - 后序遍历
- 二叉树算法题
  - 树的高度
  - 树序列化、反序列化
